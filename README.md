# Currency converter

## Quick Start

```bash
# Install dependencies for server
npm install

# Run the client
npm run client

# Run the server
npm run start

# Run the Express server with nodemon
npm run server

# Run the client and Express server both
npm run dev

# client runs on http://localhost:4200
# Server runs on http://localhost:5000
```
